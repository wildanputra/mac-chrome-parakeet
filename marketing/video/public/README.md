# Static Assets (Remotion `staticFile()`)

Remotion serves files under this directory via `staticFile()` calls
inside compositions. Three subdirectories are expected:

- **`audio/`** — voiceover WAV files generated by `npm run voice` (Kokoro)
  or `npm run voice:hq` (Higgs Audio V2). Filenames are fixed by the
  voice scripts and referenced by name in `src/compositions/Demo60.tsx`.
  Auditions land in `audio/audition/`.
- **`screencasts/`** — raw screen recordings captured from MacParakeet
  itself, ideally via Screen Studio. Filenames referenced by
  `src/content/script.ts` (`screencasts/dictation.mp4`, etc.) and by
  Demo60 / HeroLoop30. Until the files exist, `ScreencastSlot` renders
  a branded placeholder.
- **`music/`** *(optional)* — royalty-free background music for the
  demo. Not wired into compositions yet.

All three are gitignored — these are reproducible build inputs, not
source-controlled artifacts.
